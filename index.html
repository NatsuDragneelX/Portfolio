Neuron Simulation Tool

Technologies: Python, NumPy, Matplotlib, Computational Neuroscience

As part of a class project in a group of 5 to 6 people, I developed a neuron simulation tool that models and visualizes the membrane potential of neurons over time in response to different types of inputs. The tool includes:

  • Simulation Modes: Supports two modes: Current Mode and Spike Mode, simulating neuron behavior in response to different stimuli.
  • Membrane Dynamics: Uses Euler's method to solve differential equations for neuron membrane potential, with adjustable parameters.
  • Configuration File: Loads neuron model parameters from a customizable JSON configuration file.
  • Visualization: Utilizes Matplotlib to plot the evolution of membrane potential over time.
  • Command-Line Interface: Enables users to configure the simulation duration and other parameters via command-line arguments.

This project demonstrates my proficiency in mathematical modeling, scientific programming, and data visualization.

Whack-a-Mole Game

An engaging and fast-paced JavaScript game where users test their reflexes by hitting moles that randomly appear on a grid. The game features multiple modes, including:

  • Classic Mode: Players have 30 seconds to hit as many moles as possible. The game gets progressively harder as moles appear faster.
  • Ultimate Mode: A shorter, more intense game where multiple moles pop up at once and players race against a tighter time limit.
  • Timer Mode: Players have 60 seconds to rack up the highest score possible.

The game also includes a Combo System that rewards players with a score multiplier for consecutive hits without missing. This project demonstrates the use of JavaScript timing functions, random element generation, dynamic event listeners, and responsive design techniques.
